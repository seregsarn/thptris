Thptris
=======

An implementation of Tetris in the favicon of the browser window.

[You can play it online here](http://seregsarn.github.io/thptris).

Be warned: This is pretty awful code that I wrote mostly on a lunch break
at work one day. I had been talking with a coworker about procedural
favicons, and I thought to myself, "what's something cool and dynamic I
can do with a 16x16 display to play with some of these things?" A friend
of mine is a Tetris fanatic, so that sprung to mind. A short while later I
had a barely-functional implementation. I went home that night and fleshed
it out a bit more and added features like scoring and wallkicks, so it
should now work pretty much like most respectable tetris implementations.
But the code is still not very pretty.

It should also work fine in a bare browser window with no webserver running
at all, so if you so desire you can just download it and slap it in a
directory of its own to play offline.
